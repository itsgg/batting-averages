# README

## Load teams

```
rails loader:teams
```

## Load players and stints

```
rails loader:stints
```

## API endpoint

```
http://localhost:3000/players?year=<year>&team=<team name>
```

## Running the API client 

```
ruby client.rb --team "Chicago Cubs" --year 2018
```
